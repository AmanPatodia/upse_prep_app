# Testing Strategy

## Unit Tests

- Entity parsing and validation
- Repository mapping and merge logic
- Analytics calculations (accuracy, speed, weak areas)
- AI update transformation and ranking

## Widget Tests

- Login form validation
- Subject tree expansion and topic navigation
- MCQ answer selection and explanation reveal
- Current affairs bookmark/revise toggles

## Integration Tests

- Offline-first startup with cached data
- Sync flow (new current affairs + AI updates)
- Mock test attempt submission and analytics update

## Contract Tests

- API response schema checks
- version/delta compatibility checks

## Non-functional

- Performance profile with large datasets
- Accessibility checks (text scaling, contrast, semantics)
- Security checks (token storage, encrypted backup handling)
